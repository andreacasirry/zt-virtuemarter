#ZTVirtuemarter

ZTVirtuemarter is a Joomla package contain component, plugin, module.

All of extensions in this package are required VirtueMart 3.0

## Key Features

* Product Compare
* Product Wishlist
* Product QuickView
* Product Photo Zoom
* One page checkout
* Sale/Hot/New labels 
* Pre/Next Item Navigation
* Ajax Filter Price
* Deal of the day
* Best Seller Products
